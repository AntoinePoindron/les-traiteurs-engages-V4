Used for per-review rows where the count text would be misleading. Visual : ★★★★☆ 4.3 · 12 avis (when count > 0 and show_label) ☆☆☆☆☆ Aucun avis (when count == 0 and show_label) ★★★★☆ (when show_label is False) The stars are inline SVG so the project's static Tailwind doesn't have to ship a dynamic class set; one filled glyph + one outline glyph is enough. #} {% macro rating_stars(avg, count, size='w-4 h-4', show_label=True) %} {% set filled = (avg|round(0, 'common')|int) if avg is not none else 0 %} {% for i in range(1, 6) %} {% if i <= filled %} {% else %} {% endif %} {% endfor %} {% if show_label %} {% if count and count > 0 %} {{ "%.1f"|format(avg|float) }} · {{ count }} avis {% else %} Aucun avis {% endif %} {% endif %} {% endmacro %}
